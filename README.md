# 🌤️ Weather App (Flask + OpenWeather API)

A simple and beautiful weather application built using **Flask (Python)** and **OpenWeather API**.  
It allows users to check weather by city or automatically using their current location.

---

## 🌐 Live Demo

🚀 Check out the live app here:  
👉 https://weather-app-zm2h.onrender.com

A simple and beautiful weather application built using Flask...
## 🚀 Features

- 🔍 Search weather by city name
- 📍 Auto-detect current location
- 🌡️ Shows temperature, humidity, wind speed
- 🎨 Beautiful UI with CSS styling
- ⚡ Fast and simple Flask backend

---

## 🛠️ Tech Stack

- **Backend:** Flask (Python)
- **Frontend:** HTML, CSS, JavaScript
- **API:** OpenWeather API

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
```bash
git clone <your-repo-link>
cd project

### Install dependencies
pip install -r requirements.txt

### Add your API key

Go to https://openweathermap.org/api
 and get your API key.

Then open app.py and replace:

API_KEY = "YOUR_API_KEY"

### Run the app
python app.py

Open in browser:

http://127.0.0.1:5000/
